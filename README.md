# WordToNum
A short program that converts a number entered in English words to a numerical value.
⚠️ IMPORTANT: This project is very early in development and is prone to errors.
<br>Supports all real numbers

## Notes on input
* All inputs must be valid numbers in English. 
* The hundreds collumn must be between 1 and 9 inclusive ("seventeen hundred" is invalid, you must enter "one thousand and seven hundred")
* Likewise, the thousands collumn must be between 1 and 999 and millions must be between 1 and 999999
* Avoid hyphens and filler words such as 'a'
* For negative numbers, enter "minus" or "negative"
* For fractional numbers, end with "point" followed by each digit after the decimal point
